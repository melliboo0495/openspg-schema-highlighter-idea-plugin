# OpenSPG Schema Mark Language Highlighter Idea Plugin

An IDEA plugin for OpenSPG schema mark language syntax highlighting

## Plugin Card

## Features:

- Code style settings
- Color setting page
- Commenter
- Completion
- Formatter
- Line marker
- Reference
- Syntax highlighter
- Structure view
- Preview
