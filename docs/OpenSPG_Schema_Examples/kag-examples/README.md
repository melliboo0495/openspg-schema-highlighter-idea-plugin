# Download OpenSPG KAG  schema examples

## Source

- [OpenSPG KAG](https://github.com/OpenSPG/KAG/)

## Download Commands

```shell
export KAG_EXAMPLE_BASE="https://raw.githubusercontent.com/OpenSPG/KAG/refs/heads/master/kag/examples"

wget -O TwoWiki.schema     ${KAG_EXAMPLE_BASE}/2wiki/schema/TwoWiki.schema
wget -O BaiKe.schema       ${KAG_EXAMPLE_BASE}/baike/schema/BaiKe.schema
wget -O CsQa.schema        ${KAG_EXAMPLE_BASE}/csqa/schema/CsQa.schema
wget -O DomainKG.schema    ${KAG_EXAMPLE_BASE}/domain_kg/schema/DomainKG.schema
wget -O HotpotQA.schema    ${KAG_EXAMPLE_BASE}/hotpotqa/schema/HotpotQA.schema
wget -O Medicine.schema    ${KAG_EXAMPLE_BASE}/medicine/schema/Medicine.schema
wget -O Musique.schema     ${KAG_EXAMPLE_BASE}/musique/schema/MuSiQue.schema
wget -O RiskMining.schema  ${KAG_EXAMPLE_BASE}/riskmining/schema/RiskMining.schema
wget -O SupplyChain.schema ${KAG_EXAMPLE_BASE}/supplychain/schema/SupplyChain.schema

```
